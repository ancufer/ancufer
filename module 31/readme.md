### Описание учебного проекта   
В проекте реализована обработка загружаемых файлов формата <span style="font-size:18px;">***csv***</span> и построение графиков для выбранных столбцов загруженных файлов.


### Запуск приложения
* Для работы приложения испоьзуется код написанный в трёх файлах
    * **main.py**
    * **page_csv_to_df.py**
    * **page_df_statistic.py**
* Для запуска приложения нужно сохранить указанные файлы в одной папке и в этой папке в терминале запустить команду 
**streamlit run main.py**

### Структура страницы приложения
На странице есть вкладки верхнего уровня **Загрузка и формирование таблицы** и **Выбор и отображение статистических данных**.  
Вкладки и "подвкладки":
* **Загрузка и формирование таблицы**

    * ***Отображение csv файла***
    * ***Отображение полученной таблицы***
*  **Выбор и отображение статистических данных** 
    * ***Определение форматов столбцов и выбор столбцов для построения графиков***
    * ***Построение графиков***


### Обработка загруженных *csv* файлов
После выбора загружаемого файла на "подвкладке"  ***Отображение csv файла*** появляется информация о загруженных данных в текстовом формате. При этом, у пользователя есть возможность выбора параметров (разделители, кодировка) для преобразования <span style="font-size:18px;">***csv***</span> файла в таблицу. Так как есть <span style="font-size:18px;">***csv***</span> файлы, у которых в начале есть данные, не имеющие отношения к таблице, то пользователь самостоятельно определяет первую, используемую для построения, таблицы строку. Просмотр созданной таблицы возможен на отдельной вкладке ***Отображение полученной таблицы***.


### Построение графиков
Для созданной таблицы автоматически определяются форматы столбцов. При этом, на "подвкладке" ***Определение форматов столбцов и выбор столбцов для построения графиков*** у пользователя есть возможность изменить этот автоматически определённый формат. Далее на этой же подвкладке пользователь выбирает столбец или столбцы для отображения на графиках.  
Если выбран один числовой столбец, то на правой "подвкладке" ***Построение графиков*** выводится статистическая информация и строится график (гистограмма).  
Если выбраны два столбца, то на "подвкладке" ***Построение графиков*** пользвователь может выбрать тип графика, который он хочет увидеть для выбранных столбцов. 
Пользователь может выбирать любые столбцы, для построения графиков, но в случае, неверного выбора появляется сообщение о нерелевантности выбранных данных для построения графика.


### Сохранение графиков
Все строящиеся в приложении графики интерактивны, для дальнейшего использования этой интерактивности графики можно сохранить в формате <span style="font-size:18px;">***html***</span>
